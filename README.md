2. Add Hospitals info into the application along with the details asked in that registration page
3. Create a patient registration
4. There is an emergency button on the top bar and an SOS button
5. To use this feature you need to turn on the location on your device
6. This enables you to identify the nearest hospital
7. You can search for hospitals and doctors from choosing the options from the top bar
8. You can select an doctor and book appointment from the patient side page
9. On Hospital side home page you can access the appointments bar from the options in the middle of the home page where you see the booked appointments
